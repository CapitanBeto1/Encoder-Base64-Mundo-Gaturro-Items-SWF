# Encoder-Base64-Mundo-Gaturro-Items-SWF
Es un programa que de un SWF, te deja buscar nombres de items (los que empiecen con la palabra que pongas) de Mundo Gaturro y te saca un script en Base64 que te agrega esos items (se necesita Fiddler). Usa ffdec para leer el SWF y regex para encontrar los nombres.
